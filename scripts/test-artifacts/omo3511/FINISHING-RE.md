## 3. 옵션축 델타 (@1,000매 고정)

### 박 foil (OMO-3411 sweep, qty=500 기준)
- **면적 = 계단함수 + 하한**: 100/400/600mm² → **18,500(floor)**, 1,500mm² → 22,300, 4,000mm² → 33,600.
  - 현행 `ratePerMm2=14.87, intercept=0` 는 **틀림**: 100mm² 모델 1,487 vs 실원가 18,500(−17k 손해). 작은 박(로고)에서 손해.
  - 면적이 용지 cut 초과(10,000/15,000mm²)면 차단=0(per-axis 가드, OMO-3262 일치).
- 면: 전=후=22,300, **양면=41,100(≈1.84×)**.
- 종류: 금/은/로즈골드=22,300, 먹박=22,800(+500), **홀로그램=27,700(+5,400)**.
- 동판 section(신규 BKS10/보유 BKS20): 본 측정 동일(22,300) — 보유동판 setup 면제 미반영(보유동판 id 입력 필요할 가능성, 재조사 항목).
- 레이어: settingExistBakDongpan(2/3) 합산 미관측(22,300 고정) — **멀티레이어 합산 라이브 재확인 필요**.

### 형압 ap
- 면적: 10×10=29,400, 50×30=35,800, 100×100=0(cut 초과 차단). 면적의존 확인(박과 동형, 계단 추정).
- section(APS10 신규/APS20 보유동판)=동일 35,800, type(APT10 앞/APT20 뒤 돌출)=동일 → 가격무관.

### 도무송 die_cut
- 개수: 1=30,000, 2=42,400(+12,400), 4=66,100(+~12k/개) → **개당 가산**.
- 모양: 라운드/사각(DMT51)=30,000 vs **사물모양(DMT53)=47,900(+17,900)** → 모양 복잡도 가산.
- section: 전체(DMS20)=부분(DMS21)=30,000 → 가격무관.

### 타공 drilled_hole
- 개수: 1=6,100, 2=8,300(+2,200), 4=12,900(+~2.2k/개) → **개당 가산**.
- 크기: 4mm=8mm=6,100 → 가격무관.

### 귀도리 round_corner
- 종류: 한/두/세/네귀도리, 4mm/6mm **전부 5,900(@1,000매)** → 모서리수·반경 가격무관. **수량만 의존.**

### 에폭시 epoxy
- 면: 전(EPT10)=후(EPT20)=45,000, **양면(EPT30)=90,000(정확히 2×)**.

### 오시 osi / 미싱 missing
- 줄수: 1/2/3/십자 **전부 11,000(@1,000매)** → 줄수 가격무관.
- 방향(오시): 가로 OMD10=11,000 vs **세로 OMD20=9,000(−2,000)**.

### 넘버링 numbering
- NBT10/NBT20 × NBN11~13 **전부 0** (명함 GNC1001 용지 차단). 측정 불가 → **넘버링 허용 상품에서 별도 측정 필요**(자식이슈).
